# Flujos de datos y navegación

## Flujo de contenido
[Corpus socrático .me / JSON] → [NotebookLM / LLM] → [Guiones estructurados] → [Pipeline texto → audio] → [episodes.json + audio files] → [Frontend SPA]

## Flujo de navegación
Usuario entra → Inicio → (Podcast | Mapa Lagrange | Capítulos) → Nodo/Eje/Capítulo → Episodio ←→ Relaciones

## Flujo IA (Laboratorio)
Prompt estructurado → LLM (análisis crítico) → Resultado (texto) → No publicación automática → Revisión humana

Toda automatización debe respetar estos flujos y la revisión humana es obligatoria para la salida de IA.
